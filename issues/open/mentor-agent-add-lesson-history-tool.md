# Mentor Agent: Add Lesson History Query Tool

**Severity:** major
**Affected area:** Mentor Mode — Agent Toolset
**Reported by:** codebase audit

## Description

The mentor agent can generate lesson blocks and schedule lessons, but it **cannot query past lesson content or history**. When a student asks "What did we cover in the last calculus lesson?" or "Can you remind me what we learned about integration?", the agent has no tool to look up lesson records.

The `LongTermMemory` system stores session summaries (generated by LLM after each tutor session), but these are human-readable summaries meant for the mentor's context — they're not structured, queryable data. The agent cannot search across lessons, find specific topics, or reference lesson plans from past sessions.

## Expected behavior

The mentor agent should be able to:
- List recent lessons with dates, topics, and performance
- Retrieve the lesson plan and content blocks from a specific lesson
- Search for lessons covering a specific topic
- Query performance metrics from past lessons (questions asked, correct answers, time spent)

## Actual behavior

No lesson history tool exists. Long-term memory provides unstructured text summaries but no structured query capability.

## Code analysis

- `lib/features/lessons/data/repositories/lesson_repository.dart` — Hive-backed CRUD for Lesson records with all content blocks
- `lib/features/lessons/services/lesson_service.dart` — Query methods for lessons by subject/topic/date
- `lib/features/teaching/data/repositories/tutor_session_repository.dart` — Stores tutor session records with metrics
- `lib/core/services/long_term_memory.dart:46-53` — `storeSessionSummary()` / `getSessionSummary()` stores unstructured text
- `lib/features/mentor/services/tools/` — No lesson-history-related tool exists

## Suggested approach

1. **Create a new `GetLessonHistoryTool`** implementing `AgentTool`:
   ```dart
   class GetLessonHistoryTool extends AgentTool {
     name: 'get_lesson_history'
     description: 'Query past lesson records, content, and performance.'
     parameters: {
       type: 'object',
       properties: {
         subjectId: {type: 'string'},
         topicId: {type: 'string'},
         daysBack: {type: 'integer', default: 30, description: 'How far back to look'},
         limit: {type: 'integer', default: 5, description: 'Max lessons to return'},
         includeContent: {type: 'boolean', default: false, description: 'Include lesson block content'},
       },
       required: [],
     }
   }
   ```

2. **Return structured lesson data**:
   ```json
   {
     "lessons": [
       {
         "id": "lesson_001",
         "title": "Introduction to Integration",
         "subjectId": "subj_math",
         "topicId": "topic_integration",
         "date": "2026-06-28",
         "durationMinutes": 45,
         "blocksCount": 8,
         "blockTypes": ["slide", "text", "example", "exercise", "quiz", "summary"],
         "performance": {"questionsAsked": 5, "correct": 3, "accuracy": 0.6},
         "summary": "Covered basic integration rules..."
       }
     ]
   }
   ```

3. **Register and wire** through `LessonService` and `TutorSessionRepository`
